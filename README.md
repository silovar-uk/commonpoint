# 共通点のある選手を探す

浦和レッズの初めて観戦ガイドへ組み込むことを想定した、GitHub Pages用の静的プロトタイプです。

## できること

- 誕生日の月日から、同じ誕生日・同じ誕生月の選手を探す
- 出身都道府県を、ヒートマップ風の日本地図・地域ボタンから選ぶ
- 生まれ年から、同じ年・近い世代の選手を探す
- 誕生日に近いホームゲームを表示する
- 公式選手プロフィール、試合情報、観戦ガイドへ遷移する

## 公開

GitHub Pages向けの静的ファイルだけで動きます。リポジトリの **Settings → Pages** で、公開ソースを **GitHub Actions** に設定してください。

通常の公開先は `https://silovar-uk.github.io/commonpoint/` です。

## 更新する場所

- 選手情報：`data/players.js`
- 試合情報：`data/matches.js`
- デザイン：`styles-1.css` 〜 `styles-5.css`
- 表示・検索ロジック：`js/app.js`

## 公開前に必ず確認すること

- `data/players.js` の出身地・生年月日・所属状況を、公式プロフィールで全件照合する
- `data/matches.js` は **デモデータ**。試合日時、対戦相手、会場、販売状態、遷移先を必ず公式日程へ差し替える
- 公式プロフィールURL
- 選手ポートレートの追加（本プロトタイプは肖像を同梱せず、背番号グラフィックで表示）
- スタジアム名、チケットURL、試合情報URL

## 素材の扱い

先に制作した導入ビジュアル、誕生日導線、地図、一致なし表示のトーンを、公開版では軽量なCSS表現とSVGへ再構成しています。正式実装では、都道府県別に分離した正確なSVG地図と、クラブ承認済みの選手ポートレートへ差し替えてください。
