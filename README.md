# レッズ接点チェック

浦和レッズ「初めて観戦ガイド」の `GAME` と `GOURMET & GOODS` の間に差し込む、埋め込み型コンポーネントのプレビューです。

GitHub Pages上では、前後のガイド文脈を薄く再現した確認画面として表示します。独立LPとしては使いません。

## 体験コンセプト

```text
検索パーツではなく、レッズとの接点を見つけるミニ体験。
```

子どもがワクワクして押したくなり、大人の女性が見ても気分が上がり、既存ファンが見ても軽すぎない「チャーミング」なトーンを狙っています。

## できること

- 3つの入口カードを押すと、該当するフォームだけがアコーディオン展開する
  - 誕生月でつながる
  - 出身地でつながる
  - 世代でつながる
- 誕生月：月を選ぶだけで、同じ誕生月の選手・クラブマスコットを自動表示
- 出身地：都道府県と海外の出身国から検索。ヒートマップ風の分布表示と出身地選択を併用
- 世代：同じ生まれ年 → 前後1年 → 同じ年代の順で検索
- 結果冒頭に「つながり発見！」のリアクションを表示
- 3月は「レディアの誕生月！」、12月は「シャーレくん・ディアラちゃんの誕生月！」を特別表示
- クラブマスコットとして、レディア、シャーレくん、ディアラちゃんを検索対象に含める
- 結果は最大3件。4件以上は公式トップチーム一覧へ遷移
- 選手カードは公式プロフィールへ同じタブで遷移。マスコットカードはクラブマスコットとして表示
- 入力した誕生月・出身地・生まれ年は保存・送信しない

## クラブマスコットの登録

```text
レディア
1992年3月31日誕生

シャーレくん
2006年12月2日誕生

ディアラちゃん
2006年12月2日誕生
```

`data/players.js` では `type: 'mascot'` として保持しています。出身地検索の海外国チップには混ぜず、誕生月・世代検索に表示します。

## 演出方針

- 背景はアイボリー、薄いピンク、クリームを中心にして、赤黒を強くしすぎない
- 入口カードは「探す」ではなく「つながる」体験に寄せる
- 結果表示では「つながり発見！」を先に見せる
- マスコットカードは星・ドットを使い、選手カードより少し親しみやすくする
- アニメーションは入口カード・結果表示・特別メッセージに限定
- `prefers-reduced-motion` では演出を抑える

## ファイル構成

```text
index.html
  GitHub Pages用プレビュー。GAME → 接点チェック → GOURMET の文脈だけを再現

component/common-point.css
  埋め込みパーツとプレビューの基本スタイル

component/common-point-v2.css / component/common-point-v2.js
  入口カードと結果表示の演出強化

component/common-point-v3.css
  自動検索の注記、海外出身地チップ、マスコットカード、つながり発見演出のスタイル

component/common-point.js
  状態管理、アコーディオン、自動検索、結果文言、特別メッセージ、軽量計測フック

data/players.js
  選手・クラブマスコットのデータ。シーズン中の加入・退団・背番号変更時に更新

data/matches.js
  現在は未使用。公式確認済みの試合情報を扱う場合のみ追加
```

## 公式サイトへ組み込むとき

1. `component/common-point.css`、`component/common-point-v2.css`、`component/common-point-v3.css` をテーマ側で1回だけ読み込む
2. `component/common-point.js` と `component/common-point-v2.js` を `defer` または `type="module"` で1回だけ読み込む
3. `index.html` 内の `.cp-guide-player` セクションだけを、ガイドの `GAME` と `GOURMET & GOODS` の間へ移す
4. `data/players.js` を、公式情報に基づき更新する

各コンポーネントは `.js-common-point` ごとに初期化されるため、同じページに複数置いても状態が衝突しません。

## 公開前チェック

- `data/players.js` の生年月日・出身地・所属状況・公式プロフィールURLを全件確認
- クラブマスコットの誕生日・表記を確認
- 3月でレディアの特別メッセージが出る
- 12月でシャーレくん・ディアラちゃんの特別メッセージが出る
- 出身地検索にマスコットが混ざらない
- 世代検索にマスコットが出る
- 出身地の海外国名が公式プロフィールの表記と合っているか確認
- 選手写真を追加する場合は、クラブ承認済み素材だけを使用
- 375px幅のスマホ、Safari iOS、キーボード操作、モーション軽減設定を確認
